Wireguard Easy with PiHole
===========================

## Installation

* **Clone**
* **Install Docker**
````
curl -fsSL https://get.docker.com -o get-docker.sh  
sudo sh get-docker.sh
````
* **Install wg-easy and pihole**  
````
cd wg-easy-pihole
docker compose up
````
